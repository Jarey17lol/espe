# Taller 1 big data

Este repositorio contiene la solución del taller práctico de SQL desarrollado sobre la base de datos tienda.db.

El objetivo fue analizar la información de clientes, productos y ventas de la tienda “El Último Salto”, respondiendo 30 preguntas de negocio mediante consultas SQL estructuradas.

Durante el desarrollo se aplicaron sentencias SELECT, funciones de agregación, filtros con WHERE, JOIN, subconsultas y CTE (WITH).

# Desarrollo del Taller

Los 30 ejercicios abarcan desde consultas básicas hasta análisis más avanzados sobre la base de datos.

Se realizaron:

Consultas simples de selección, filtrado y ordenamiento.

Cálculos con funciones agregadas como COUNT(), SUM(), AVG(), MAX() y MIN().

Análisis de ingresos, productos más vendidos y métricas mensuales.

Relaciones entre tablas mediante JOIN.

Subconsultas para segmentación de clientes.

Uso de CTE (WITH) para análisis estructurados.

Cálculo de porcentajes, diferencias entre periodos y clasificación por rangos de edad.

El enfoque fue completamente analítico, trabajando únicamente con consultas SELECT, sin modificar la base de datos.
